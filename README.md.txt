# Analisis Simulatif Data: Curah Hujan dan Muka Air Tanah di Area Tambang Terbuka

## Deskripsi Proyek

Proyek ini merupakan latihan analisis data simulatif menggunakan bahasa SQL untuk mengeksplorasi hubungan antara curah hujan dan muka air tanah (piezometer) di area tambang terbuka. Studi ini bertujuan melatih keterampilan analisis data dengan pendekatan praktis dan sederhana.

## Tujuan

- Melatih penggunaan SQL dalam menganalisis data lingkungan tambang
- Menunjukkan potensi korelasi antara curah hujan dan perubahan muka air tanah
- Membuat portofolio awal untuk transisi karier ke data analyst

## Tools yang Digunakan

- SQLite (via DB Browser for SQLite)
- GitHub Desktop
- Microsoft Word & PDF
- Canva (untuk visualisasi dan desain)

## Langkah-Langkah Analisis

1. Membuat data simulatif curah hujan dan piezometer berdasarkan skenario realistis di tambang terbuka
2. Mengimpor data ke dalam database SQLite
3. Menjalankan kueri SQL untuk:
   - Melihat sebaran data
   - Menyaring data berdasarkan kriteria tertentu
   - Menggabungkan data untuk melihat kemungkinan hubungan
4. Menyusun laporan analisis dalam bentuk dokumen

## Cara Menjalankan

- Unduh file database `.db` (jika tersedia) atau gunakan struktur tabel yang ditunjukkan dalam dokumen
- Buka dengan DB Browser for SQLite
- Jalankan kueri yang tersedia di bagian Lampiran dokumen PDF

## Hasil Akhir

Hasil analisis ditampilkan dalam laporan PDF yang mencakup penjelasan, kueri SQL, dan interpretasi data. Pendekatan yang digunakan masih sederhana namun dapat menunjukkan kemampuan dasar dalam eksplorasi dan pemfilteran data.

## Catatan

Proyek ini bersifat simulatif dan tidak menggunakan data aktual. Semua data dibuat untuk keperluan belajar dan portofolio.
